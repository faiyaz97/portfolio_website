Arduino home automation

A couple of years ago I worked on a project based on my knowledge and interest in Html and Arduino. My aim was to create a home automation system through the Ardunino microcontroller and various sensors and motors. At first I designed the project so that I could have a theoretical view of it. Then I code it and proceeded by creating an interface to send instructions to Arduino through a website so that I could have turned on or off the light, read the temperature in the room, lock the door and much more.

Hotel management with Access

I also worked with databases. One of my recent projects was about a hotel management through Microsoft Access. The latter is really useful in terms of storing a great amount of data and the goal of this project was to make an application that allowed to handle all hotel activities. Through this database on Microsoft Access, it was possible to check the availability of the rooms, make bookings or add more services in the booking such as breakfast, lunch or dinner as well as checking the prices of the rooms.

School enrolment with Java

A project that I worked for in college was the java project on school enrolment system. Its main objective was to manage students' details. Practically what I programmed was an interface to insert the students' personal data and a programme which could check if that student was eligible for one of the course we had at college ( Access course). If the student met the criteria he or she could choose their major subjects and check their lesson's timetable.
